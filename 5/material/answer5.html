総合問題 — Final Challenge

items.zip を調査し、3つのファイルに隠されたフラグの断片を集めて組み合わせよ

ディレクトリ5の items.zip には app・image・invoice.pdf の3ファイルが入っています。それぞれ異なる仕掛けが施されており、各ファイルにフラグの断片が隠されています。これまでの章で学んだすべての技術を使って、フラグを完成させてください。

答えを見る
kindai{surface_analysis_master}

3つのファイルから断片を取り出して繋げるとフラグが完成します。

Step 0 — 展開して形式を確認
unzip items.zip -d items && cd items
file *
# app      → data
# image    → data
# invoice.pdf → JPEG image data（拡張子と中身が違う！）
app
XORエンコード＋UPXパック
断片①：kindai{surface_
image
先頭4バイト破損 PNG
断片②：analysis_
invoice.pdf
実は JPEG
断片③：master}
Step 1 — app からフラグ断片を取る（第4章の手法）
strings app   # XORで難読化されており文字列は読めない
upx -d app    # UPXアンパック
./app         # 実行すると断片が表示される
# → kindai{surface_
Step 2 — image からフラグ断片を取る（第3章の手法）
xxd image | head -1
# 00 00 00 00 0D 0A 1A 0A  ← PNG の先頭4バイトが破損している

( printf '\x89\x50\x4E\x47'; tail -c +5 image ) > image_fixed.png
file image_fixed.png   # → PNG image data
open image_fixed.png   # 画像内に断片が描かれている
# → analysis_
Step 3 — invoice.pdf からフラグ断片を取る（第1章の手法）
file invoice.pdf   # → JPEG image data（PDF ではない！）

cp invoice.pdf fake.jpg
open fake.jpg   # 画像を開くと断片が見える
# → master}

断片を繋げる： kindai{surface_ + analysis_ + master}

UPXアンパック・シグネチャ修復・拡張子偽装の看破という全技術のフルコース。
